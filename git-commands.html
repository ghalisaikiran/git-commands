Unstage added files: git restore --staged [FILENAME.extension]

Restore to a previous commit: git restore [commit]

Stash uncommitted changes: git stash

Unstash uncommitted changes: git stash pop

Link your local project to a git repository: git remote add origin [REPOSITORY_URL]

If a branch does not exists in git repository and you want to create branch and push it: git push --set-upstream origin [BRANCH_NAME]

How to revert code in repository without losing it? 1. Go to previous commit: git reset [PREV_COMMIT] 2. Stash the commit that is causing issue, git stash 3. Now, force push the prev commit to repository: git push -f 4. Now, get back our changes in local: git stash pop.

Fetch all branches: git fetch

Fetch all branches including deleted branches: git fetch -all -prune

Merge all un pushed commits into a single commit - 1. git reset --soft HEAD~2 (number of commits to be merged) 2. git commit -m "new commit message" 3. git push -f
